## Interview prep study plan prompt example

> Before using this prompt, replace the bracketed placeholders with your own details. Feel free to add more context and customize to your liking — the more specific you are about your stack, experience level, and goals, the better the plan will be.

```
I want to prepare for a technical interview. My stack is [e.g. Python/Django/PostgreSQL]. I have [X weeks / full time / evenings only] available to study.

Generate a detailed interview prep study plan as a markdown file with the following structure:
- Organized by phase and week, with each week mapping to a specific set of topics
- Each topic has all necessary reading links listed before the exercises
- Each exercise is fully solvable from the linked reading material — no gaps
- Everything is a checkbox (both reading and exercises)
- Each day is split into morning (theory/reading) and afternoon (exercises/building)
- Saturday is a mock session
- Links must be to free resources
- Where a single link doesn't cover everything needed for an exercise, provide multiple links

My goals are: [e.g. be prepared for both frontend and backend, theoretical questions and live coding]
```
